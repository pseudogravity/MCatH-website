Tutorial - How to load the seed in Alpha
Make sure you're using Minecraft Java Edition, with "Historical versions" enabled in the launcher, and choose version a1.0.16. This version doesn't support entering seeds, so you'll either have to use save file editing, or just use our already prepared world file:
https://files.mcatho.me/herobrine/World1.zip
This will generate a fresh version of the herobrine world, with you standing at the exact coordinates. Make sure to use the correct Minecraft version or it won't work.
If you have any problems, ask us on our Discord.
